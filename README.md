# 🛒 E-Commerce Web Application

A full-stack E-Commerce web application built using Node.js, Express.js, MongoDB Atlas, HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://ecommerce-ush9.onrender.com

## 📂 GitHub Repository

https://github.com/ffrshdd/ecommerce

## ✨ Features

- User Registration
- User Login
- JWT Authentication
- Admin & User Roles
- Product Management (CRUD)
- Shopping Cart
- Checkout
- Order Management
- Responsive Web Interface
- MongoDB Atlas Database

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

### Authentication
- JSON Web Token (JWT)
- bcryptjs

## ⚙️ Installation

```bash
git clone https://github.com/ffrshdd/ecommerce.git

cd ecommerce

npm install
```

Create a `.env` file:

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_CONNECTION_STRING
JWT_SECRET=YOUR_SECRET_KEY
```

Run the application:

```bash
npm run dev
```

Open:

```
http://localhost:5000
```
